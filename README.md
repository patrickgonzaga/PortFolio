# Developer Portfolio 🚀

A high-end, responsive developer portfolio built with a Nordic minimalist aesthetic, featuring smooth animations and a focus on professional narrative. This project showcases over 15 years of experience in software engineering and cloud architecture.

![Portfolio Preview](https://raw.githubusercontent.com/lucide-react/lucide/main/icons/layout.svg) <!-- Replace with actual screenshot link -->

## ✨ Features

-   **Nordic Minimalist Design**: Clean, sophisticated, and narrative-driven UI.
-   **Interactive Experience**: Engaging animations and transitions powered by Framer Motion.
-   **Modular Architecture**: Built with highly reusable components following SOLID principles.
-   **Dynamic Content**: Centralized data management via `cvData.ts` for easy updates.
-   **Responsive Layout**: Fully optimized for mobile, tablet, and desktop views.
-   **Core Sections**:
    -   **Hero**: Dynamic introduction with personal branding.
    -   **Expertise Grid**: Visual representation of technical skills and domains.
    -   **Experience Timeline**: Interactive professional journey.
    -   **Projects**: Clickable project cards with detailed modal views.
    -   **Certifications**: Showcasing industry-recognized credentials.
    -   **Contact**: Integrated social links and professional contact details.

## 🚀 Tech Stack

-   **Framework**: [React 19](https://react.dev/)
-   **Build Tool**: [Vite 8](https://vitejs.dev/)
-   **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
-   **Animations**: [Framer Motion 12](https://www.framer.com/motion/)
-   **Language**: [TypeScript 6](https://www.typescriptlang.org/)
-   **Icons**: [Lucide React](https://lucide.dev/)

## 🛠️ Getting Started

### Prerequisites

-   Node.js 20.x or later
-   pnpm, npm, or yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies**:
    ```bash
    pnpm install
    ```

3.  **Run the development server**:
    ```bash
    pnpm dev
    ```

4.  **Open the app**:
    Navigate to [http://localhost:5173](http://localhost:5173).

## 📁 Project Structure

```text
src/
├── components/
│   ├── sections/    # Main sections (Hero, Projects, etc.)
│   └── ui/          # Atomic UI components (Buttons, Modals, Cards)
├── data/            # cvData.ts and other static data sources
├── contexts/        # React Contexts for global state (Theme, etc.)
├── assets/          # Images, logos, and fonts
├── App.tsx          # Main entry component
├── index.css        # Global styles and Tailwind imports
└── main.tsx         # React DOM rendering
```

## 🎨 Design Principles

-   **High Contrast**: Leveraging sleek dark modes and subtle accents.
-   **Micro-animations**: Enhancing user engagement without distracting from the content.
-   **Typography**: Using premium sans-serif fonts for readability and style.
-   **Consistency**: Unified design language across all components.
